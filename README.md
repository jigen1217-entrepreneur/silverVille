# 🏡 SilverVille: Mind Oasis

> 액티브 시니어용 치매 예방 의학 논문 기반 시니어 소셜 경영 시뮬레이션 게임

## 핵심 시스템

- 🥗 **AI 식단 카메라** — MIND 식단 인식 → 마법의 비료 보상
- 🚶 **이중 과제 산책** — 만보기 + 음성 퀴즈 동시 수행
- ☕ **귀 밝은 바리스타** — 청각 + 작업기억 훈련 미니게임
- 💌 **가족 연동 우체통** — 카카오톡 연동 칭찬 시스템

## 기술 스택

- **Client:** React Native (Expo) + TypeScript + Zustand
- **Server:** Node.js + Express + Prisma + PostgreSQL
- **AI:** OpenAI Vision API
- **Infra:** AWS (ECS + RDS + S3)

## 시작하기

```bash
npm install
npx expo start
```
