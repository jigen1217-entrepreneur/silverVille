## 5. Senior Accessibility Considerations

### Visual

| Guideline | Implementation | Compliance |
|-----------|----------------|------------|
| Min font size | 16px body, 18px buttons | WCAG AA |
| Contrast ratio | All text/bg combos >= 4.5:1 | WCAG AA |
| Touch targets | Min 56x56dp (SeniorButton) | Exceeds WCAG (48dp) |
| Color not sole indicator | Icons + text for all states | WCAG 1.4.1 |
| System font scaling | Test at "Largest" setting | No truncation |

### Auditory

| Guideline | Implementation |
|-----------|----------------|
| TTS speed | Default 0.85x; configurable 0.7x-1.0x |
| Replay button | Every TTS element has visible replay |
| Visual fallback | All TTS content shown as text simultaneously |
| Audio feedback | Gentle sounds for correct/wrong (optional) |

### Motor

| Guideline | Implementation |
|-----------|----------------|
| No long press | All actions via single tap |
| No swipe gestures | Scroll only; no swipe-to-delete |
| Undo support | Confirm dialog before destructive actions |
| Timeout lenient | No auto-dismiss under 10 seconds |

### Cognitive

| Guideline | Implementation |
|-----------|----------------|
| Simple navigation | 5 clearly labeled tabs, no nested menus |
| Consistent layout | Same header/content/action pattern per screen |
| Encouraging language | Positive feedback even on low scores |
| No jargon | All text in simple Korean |
